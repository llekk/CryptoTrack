# CryptoTrack

**CryptoTrack** — настольное приложение на C++ и Qt для отслеживания и анализа криптовалют.  
Приложение получает рыночные данные через CoinGecko API, отображает текущие цены, историю курса, графики, технические индикаторы и позволяет создавать ценовые оповещения.

Проект разработан как курсовая работа по программированию.

---

## Основные возможности

- отображение 7 популярных криптовалют:
  - BTC — Bitcoin;
  - ETH — Ethereum;
  - SOL — Solana;
  - DOGE — Dogecoin;
  - ADA — Cardano;
  - XRP — Ripple;
  - DOT — Polkadot;
- загрузка актуальных цен через CoinGecko API;
- отображение изменения цены за 24 часа;
- выбор периода анализа:
  - 24 часа;
  - 7 дней;
  - 30 дней;
  - 90 дней;
- линейный график цены;
- свечной график `Open / High / Low / Close`;
- отображение объёмов торгов;
- расчёт и отображение RSI;
- уровни поддержки и сопротивления;
- возможность включать и выключать уровни на графике;
- конвертор криптовалют;
- добавление ценовых оповещений;
- автоматическая проверка оповещений при обновлении данных;
- удаление выбранных оповещений;
- очистка сработавших оповещений;
- сохранение оповещений в `alerts.json`;
- сохранение пользовательских настроек в `settings.json`;
- автообновление данных;
- обработка сетевых ошибок и лимитов API.

## Интерфейс приложения

Интерфейс разделён на три основные зоны.

### Левая панель

Содержит список криптовалют.  
Для каждой монеты отображается:

- тикер;
- название;
- текущая цена;
- изменение за 24 часа;
- цветовая индикация роста или падения.

### Центральная область

Отображает подробную информацию по выбранной монете:

- название и тикер;
- текущую цену;
- изменение за 24 часа;
- кнопки выбора периода;
- переключение между линейным и свечным графиком;
- основной график цены;
- объёмы торгов;
- RSI;
- уровни поддержки и сопротивления.

### Правая панель

Содержит:

- блок создания ценовых оповещений;
- список активных и сработавших оповещений;
- кнопки удаления и очистки оповещений;
- конвертор криптовалют.

## Работа с данными

Приложение использует CoinGecko API.

Для загрузки текущих цен используется запрос к рынку криптовалют.  
Для истории цены используется исторический график выбранной монеты.

Приложение учитывает возможные ограничения API.  
Если CoinGecko временно ограничивает количество запросов, приложение не завершается аварийно, а выводит сообщение о необходимости подождать.

## Графики

Графики строятся вручную с использованием `QPainter`.

Реализованы следующие типы отображения:

### Линейный график

Показывает изменение цены выбранной монеты за указанный период.

### Свечной график

Каждая свеча содержит:

- `Open` — цена открытия;
- `High` — максимальная цена внутри интервала;
- `Low` — минимальная цена внутри интервала;
- `Close` — цена закрытия.

Свечи строятся на основе исторических цен, полученных через API.

### Объёмы торгов

Отображаются в виде столбчатой диаграммы.

### RSI

RSI рассчитывается по ценам закрытия.  
На графике отображаются уровни 30 и 70, которые используются для оценки зон перепроданности и перекупленности.

---

## Уровни поддержки и сопротивления

Приложение рассчитывает уровни поддержки и сопротивления на основе локальных минимумов и максимумов цены.

- Поддержка — область, где цена ранее разворачивалась вверх.
- Сопротивление — область, где цена ранее разворачивалась вниз.

Пользователь может включать и выключать отображение этих уровней кнопкой `Уровни`.

---

## Оповещения

Пользователь может создать оповещение по цене:

- выбрать монету;
- выбрать условие: цена выше или ниже;
- указать целевую цену.

При обновлении данных приложение проверяет все активные оповещения.  
Если условие выполнено, появляется уведомление, а оповещение получает статус `сработало`.
